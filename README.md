# valorant.js

### Unofficial Valorant api wrapper

Riotgames 발로란트의 전적 API 일부를 래핑하는 라이브러리입니다.

## Use

공식적인 API가 아닌 전적조회 서비스 **[dak.gg](https://dak.gg/)** 플랫폼의 API를 리버싱하여 제작된 라이브러리 입니다.  
**사용시 발생하는 불이익과 법적 책임은 모두 사용자에게 있습니다.**
